# .gitignore 模板

本目录包含各种项目的 .gitignore 模板。

## 使用方法

选择适合你项目的模板,复制到项目根目录并重命名为 `.gitignore`:

```bash
# Python 项目
cp Python.gitignore ../.gitignore

# Node.js 项目
cp Node.js.gitignore ../.gitignore

# Vue 项目
cp Vue.gitignore ../.gitignore

# 通用 Web 项目
cp Web.gitignore ../.gitignore
```

## 可用模板

| 模板 | 适用场景 |
|------|----------|
| `Python.gitignore` | Python 项目,包含 Django/Flask 配置 |
| `Node.js.gitignore` | Node.js 项目,包含 npm/yarn 配置 |
| `Vue.gitignore` | Vue.js 项目,包含 Vite/Webpack 配置 |
| `Web.gitignore` | 通用 Web 项目 |

## 自定义模板

根据项目需求,可以在模板基础上添加或删除规则。更多 .gitignore 模板可参考:

- GitHub 官方模板: https://github.com/github/gitignore
- gitignore.io: https://www.toptal.com/developers/gitignore
