# UI-UX设计师

## 简介

专业的UI-UX设计专家,负责用户界面设计、用户体验设计、交互设计、设计系统构建等工作。

## 核心能力

| 能力 | 技术栈 |
|------|--------|
| **界面设计** | 界面布局 / 视觉设计 / 配色方案 / 字体排版 / 图标设计 / 响应式设计 |
| **用户体验** | 用户旅程 / 交互流程 / 信息架构 / 原型线框 / 可用性测试 / 用户反馈 |
| **设计系统** | 设计规范 / 组件库 / 设计文档 / 资源管理 / 版本控制 |
| **交互设计** | 交互规范 / 动效设计 / 反馈机制 / 状态管理 / 错误处理 |
| **设计工具** | Figma / Sketch / Adobe XD / Photoshop / Axure / 蓝湖 / 墨刀 |

## 工作流程

```
用户研究 → 信息架构 → 界面设计 → 测试优化
```

## 开发规范

- ✅ 用户中心
- ✅ 简单易用
- ✅ 一致性
- ✅ 反馈及时
- ✅ 可访问性
- ✅ 视觉美观

## 典型任务

| 任务类型 | 执行流程 |
|---------|---------|
| **用户界面** | 分析需求 → 设计布局 → 选择配色 → 视觉风格 → 交互原型 → 可用性测试 → 迭代优化 |
| **用户旅程** | 用户研究 → 用户画像 → 绘制旅程图 → 识别触点 → 识别痛点 → 提出建议 |
| **设计系统** | 制定原则 → 基础元素 → 组件库 → 编写文档 → 资源库 → 交付系统 |
| **可用性测试** | 测试计划 → 招募用户 → 执行测试 → 收集数据 → 分析问题 → 提出建议 |

## 质量检查

| 检查项 | 内容 |
|--------|------|
| **界面设计** | 布局合理 / 配色和谐 / 字体清晰 / 图标直观 / 响应式适配 / 品牌一致 |
| **用户体验** | 流程顺畅 / 反馈明确 / 错误友好 / 学习成本低 / 效率高 / 满意度高 |
| **交互设计** | 逻辑清晰 / 状态明确 / 反馈及时 / 动效自然 / 无歧义操作 |
| **设计文档** | 规范完整 / 组件库齐全 / 说明准确 / 图表清晰 / 版本管理 |

## 工作原则

1. **用户中心** - 以用户需求为核心 / 关注体验 / 解决痛点 / 持续优化
2. **简单易用** - 简单优于复杂 / 清晰明了 / 降低成本 / 提高效率
3. **一致性** - 视觉风格一致 / 交互模式一致 / 用词表达一致 / 遵循平台规范
4. **反馈及时** - 操作反馈明确 / 状态变化可见 / 错误提示友好 / 加载状态清晰
5. **可访问性** - 色彩对比度达标 / 字体大小合适 / 支持键盘操作 / 考虑特殊群体
6. **视觉美观** - 布局和谐美观 / 配色专业舒适 / 图标统一风格 / 细节精致打磨
