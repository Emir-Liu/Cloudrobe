# Cloudrobe 云衣橱 - 用户故事 (User Stories)

## 1. 用户故事概述

### 1.1 用户故事格式
```
作为 [角色],
我希望 [功能/需求],
以便 [价值/目标]
```

### 1.2 故事优先级（MoSCoW）
- **Must have（必须有）**：MVP核心功能，必须实现
- **Should have（应该有）**：重要功能，第一版后续实现
- **Could have（可以有）**：锦上添花，有时间再做
- **Won't have（不做）**：当前阶段不考虑

## 2. 用户认证模块

### US-001: 手机号注册/登录
**优先级**: Must have

**用户故事**:
```
作为新用户,
我希望通过手机号+验证码注册/登录,
以便快速开始使用云衣橱服务
```

**验收标准**:
- [ ] 输入手机号后点击"获取验证码"
- [ ] 60秒倒计时，期间不能重复获取
- [ ] 验证码6位数字，5分钟内有效
- [ ] 验证通过后自动登录或注册
- [ ] 手机号格式校验（11位，1开头）
- [ ] 验证码错误提示
- [ ] 验证码过期提示

**故事点**: 3
**依赖**: 无

## 3. 衣物发布模块

### US-005: 发布衣物
**优先级**: Must have

**用户故事**:
```
作为出租方,
我希望发布自己的衣物信息,
以便让其他用户浏览和租赁
```

**验收标准**:
- [ ] 上传3-5张清晰照片（正面、背面、细节）
- [ ] 填写衣物名称（必填）
- [ ] 选择品牌（必填）
- [ ] 选择分类（必填）
- [ ] 选择尺码（必填）
- [ ] 选择新旧程度（必填）
- [ ] 设置日租金（必填，≥1元）
- [ ] 设置押金（必填，≥10元）
- [ ] 设置最短租期（必填，≥1天）
- [ ] 设置最长租期（必填，≤30天）
- [ ] 选择配送方式（必填）
- [ ] 填写衣物描述（可选）
- [ ] 点击"发布"按钮
- [ ] 发布成功后跳转到"我的衣橱"
- [ ] 未实名认证时提示先认证

**故事点**: 8
**依赖**: US-003
