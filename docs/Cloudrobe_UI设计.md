# Cloudrobe 云衣橱 - UI设计文档

## 1. 设计概述

### 1.1 设计理念
- **时尚简约**：符合服装行业调性，突出产品本身
- **用户友好**：操作流程清晰，减少用户认知负担
- **视觉层次**：通过色彩、字体、间距建立清晰的视觉层级
- **情感共鸣**：使用柔和的粉色系，营造温暖、信任的氛围

### 1.2 目标用户
- 25-40岁女性为主
- 追求时尚，注重生活品质
- 熟悉移动互联网操作

## 5. 交互设计规范

### 5.1 页面转场

#### Push转场（页面入栈）
- 效果：新页面从右侧滑入
- 时长：300ms
- 缓动：ease-out
- 阴影：页面左侧有阴影

#### Pop转场（页面出栈）
- 效果：当前页面向右侧滑出
- 时长：300ms
- 缓动：ease-out

#### Modal弹窗
- 效果：从底部向上滑入
- 背景遮罩：黑色，透明度 0.5
- 最大高度：屏幕 80%
- 圆角：顶部 16px

### 5.2 反馈机制

#### Loading加载
- 菊花图：白色或黑色
- 大小：32px
- 文字：加载中...，14px
- 背景：白色，圆角 8px

#### Toast提示
```
┌────────────────┐
│     ✓ 成功     │ 2秒
└────────────────┘

┌────────────────┐
│     ! 错误     │ 3秒
└────────────────┘
```
- 背景：黑色，透明度 0.8
- 文字：白色，14px
- 内边距：12px 24px
- 圆角：4px
- 位置：屏幕底部居中

#### Dialog确认弹窗
```
┌──────────────────────┐
│       提示            │ 标题: 18px Bold
├──────────────────────┤
│                      │
│  确认要删除此衣物吗？ │ 内容: 14px
│                      │
├──────────────────────┤
│   [取消]      [确认]  │ 按钮: 16px Medium
└──────────────────────┘
```
- 圆角：12px
- 宽度：最大 320px
- 按钮：左灰色，右粉色

### 5.3 手势操作

#### 下拉刷新
- 触发区域：页面顶部
- 刷新图标：菊花图
- 提示文字：
  - 下拉刷新...
  - 释放刷新...
  - 正在刷新...
  - 刷新成功

#### 上拉加载
- 触发区域：页面底部
- 加载图标：菊花图
- 提示文字：
  - 上拉加载更多...
  - 释放加载更多...
  - 正在加载...
  - 没有更多了

#### 左滑操作
- 应用：订单列表
- 操作按钮：
  - 删除：红色背景
  - 取消订单：灰色背景
- 宽度：80px
- 动画：300ms ease-out

#### 图片预览
- 触发：点击图片
- 效果：从点击位置放大至全屏
- 操作：
  - 双指缩放
  - 单指拖动
  - 左右滑动切换
- 关闭：点击背景或缩小

### 5.4 按钮交互

#### 点击效果
- 主按钮：缩放至 0.96，透明度 0.9
- 次按钮：背景变暗
- 文本按钮：透明度 0.7
- 持续时间：150ms

#### 长按效果
- 菜单弹出
- 震动反馈
- 显示操作菜单

#### 禁用状态
- 背景灰色 #E5E5E5
- 文字灰色 #999999
- 无点击效果

### 5.5 表单交互

#### 输入框
- 聚焦：边框变粉红 #FF6B9D
- 输入：实时显示字符数
- 清除：输入后显示 × 按钮
- 校验：失去焦点后校验

#### 选择器
- 点击：弹出选择器
- 选中：高亮显示
- 确认：更新显示
- 取消：保持原值

#### 开关
- 状态：开 / 关
- 颜色：开 #FF6B9D，关 #E5E5E5
- 动画：300ms 滑动
